# Settings

Sure provides various settings to customize your experience and control how the app behaves.

## Security settings

### High-resolution logos

Sure can fetch high-resolution logos for merchants and financial institutions using the Brandfetch service. This feature enhances the visual appearance of your transactions and accounts.

**To enable high-resolution logos:**

1. Navigate to Settings → Security
2. Scroll to the "Brand Fetch Settings" section
3. Toggle "Use high-resolution logos" to enable
4. The setting saves automatically

When enabled, Sure will:
- Fetch higher quality logos for merchants and institutions
- Use Brandfetch as a fallback for stock tickers when logos aren't available
- Display clearer, more professional-looking icons throughout the app

<Note>
High-resolution logos may increase data usage slightly as larger image files are downloaded.
</Note>

### Brandfetch integration

The Brandfetch integration provides:
- **High-quality logos**: Professional brand assets for better visual recognition
- **Fallback support**: Automatic logo lookup for stock tickers and merchants
- **Consistent branding**: Standardized logo appearance across all transactions

This feature works automatically once enabled - no additional configuration required.

## Provider settings

Configure third-party integrations and data providers from Settings → Providers. See the [providers documentation](/providers/overview) for detailed setup instructions.

## AI settings

Configure AI features like chat and transaction rules from Settings → Self-Hosting. See the [LLM support documentation](/llm-support) for more details.

## Mobile app settings

### Theme selection

The Sure mobile app supports multiple theme options to match your preferences:

- **Light mode**: Bright interface optimized for daytime use
- **Dark mode**: Dark interface that's easier on the eyes in low-light conditions
- **System**: Automatically matches your device's system theme setting

To change the theme:

1. Open the mobile app
2. Navigate to Settings
3. Select **Theme**
4. Choose Light, Dark, or System
5. The theme updates immediately

The app remembers your preference across sessions.

### AI chat improvements

The mobile app includes an enhanced AI chat experience:

- **Typing indicator**: An animated indicator shows when the AI is generating a response
- **Smooth animations**: The typing indicator uses a classic three-dot bounce animation
- **Auto-scroll**: The chat automatically scrolls to show new messages as they arrive
- **Improved flow**: Chat creation is now lazy-loaded on first message send, eliminating loading delays

## Account preferences

Manage your personal account settings, including:
- Email and password
- Display preferences
- Notification settings
- Connected SSO accounts

Access these from Settings → Account or Settings → Security.
